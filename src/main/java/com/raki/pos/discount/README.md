# Discounts

The Discounts feature allows a business to define, manage, and monitor promotional price reductions that apply to products and, where relevant, services. It gives owners and managers a central place to configure how and when discounts are used in the system.

## What this feature does

- Shows a list of all existing discounts for the current business.
- Indicates for each discount:
  - Name
  - Type and value (percentage or fixed amount)
  - Time period (start and end date, or no expiration)
  - Current status (ACTIVE, SCHEDULED, EXPIRED, INACTIVE)
- Allows creating new discounts through a dedicated form.
- Allows editing existing discounts to change their rules and validity.
- Allows deleting discounts that are no longer needed.

## How discounts are configured

From the discount form, the user can:

- Set a **name** so the discount can be easily recognized.
- Choose where the discount applies:
  - Products
  - Services (for service businesses, such as hairdressers)
  - Both products and services
- Decide the **value**:
  - Percentage (e.g., 10% off)
  - Fixed amount (e.g., 5.00 off)
- Define the **time period**:
  - Start date
  - End date, or no expiration date
- Choose the **status** (active or inactive).
- Select specific products and/or services to which the discount will be applied.

## How discounts appear in the list

In the discounts overview table, each discount entry shows:

- The discount name.
- The type and value (e.g., “10% off” or “5.00 off”).
- The active time window (dates or “No expiration”).
- A colored status badge (e.g., ACTIVE, SCHEDULED, EXPIRED, INACTIVE).
- Action buttons for editing and deleting the discount.

If there are no discounts yet, the page clearly indicates that no discounts exist.

## When this feature is used

- When a business wants to run promotions on specific products or services.
- When the business wants to launch limited‑time offers (e.g., seasonal discounts).
- When existing discounts need to be extended, modified, or turned off.
- When old discounts should be removed so they no longer affect pricing.

## Who typically uses it

- Business owners and managers who are responsible for pricing and promotions.
- Not intended for regular employees or customers, but their actions (orders, reservations) will be affected by the discounts configured here.
