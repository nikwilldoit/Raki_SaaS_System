# Software Design Project – SaaS System for SMEs

## Project Overview
This project is part of the **Software Design course** at **Vilnius University Erasmus program**.  
We are creating a **SaaS system** for small and medium businesses in **catering** (bars, cafes, restaurants) and **beauty** (barbers, hairdressers, SPA) sectors.

The system allows employees to manage **orders, reservations, payments**, and related operations.

---

## Features

### Orders
- Create, modify, cancel orders  
- Apply discounts and tips  
- Split payments and generate receipts  
- Stripe integration for card payments  

### Reservations
- Book, modify, cancel appointments  
- Assign appointments to employees  

### System Management
- Manage users, products, services, taxes, discounts  
- Preserve historical records for audit  

### Bonus
- Inventory management  
- SMS notifications for reservations
